二、概念介绍

各个区介绍

git 分为以下三个区：

  • 工作区：这里都是没有添加到 git 跟踪的代码
  • 暂存区：已添加到 git 跟踪，可以提交到本地
  • 版本区：已提交的代码，可以推送到远程仓库

git 命令行中符号介绍：

  • + 增加
  • ~ 修改
  • - 删除

如 +2 表示有两个添加
