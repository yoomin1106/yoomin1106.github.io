字符串与数字相关函数

字符串与数字相关的函数，我自己在项目中很少用到，因此不做详述，只是列出功能，有需要的读者研究一二。

  • unquote() 去掉字符串头尾的双引号
  • quote() 给字符串头尾添加双引号
  • To-upper-case() 将字符串所有字母转换成大写
  • To-lower-case() 将字符串所有字母转换成小写
  • percentage() 将一个不带单位的数转换成百分比值
  • round() 将数值四舍五入，可以带单位
  • ceil() 向上取整，可以带单位
  • floor() 向下取整，即去除小数部分，可以带单位
  • abs() 取绝对值
  • min($number1,$number2,...) 取最小值
  • max($number1,$number2,...) 取最大值
