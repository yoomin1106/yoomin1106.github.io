四、高级选择器

层次选择器
  • $('#box p')和$('#box').find('p')是等价的
  • $('#box>p')和$('#box').children('p')也是等价的，children()是和子选择器等价的方法
  • $('#box + p')和$('#box').next('p')是等价的，prev()则相反
  • $('#box ~ p')和$('#box').nextAll('p')是等价的,表示后面所有的同级节点,prevAll()相反
  • sibling()表示所有同级元素，相当于nextAll()和prevAll()合起来
    注意：next()等价于next(‘*’),耗资源，建议少用

其他方法：

示例 说明
nextUntil(‘p’) 选择后面的兄弟元素（即同级）不是P的元素，遇到P则停止
prevUntil(‘p) 与上面相反

选择器快慢比较：

语句 说明
$(‘#box’).find(‘p’) 最快，使用getElementById,ByName,ByTagName和querySlectorAll()
$(‘#box’).children(‘p’) 会使用\$.slibing()和javascript的nextSibling()方法，慢50%
$(‘#box > p’) jQuery内部使用Sizzle引擎，从右到左解析，慢约70%
$(‘#box p’) 遍历多级，速度更慢，慢77%

综上，find()最快，$(‘#box p’)这种高级选择器最慢，先用var a=$('#box')会先进行缓存，提高性能

属性选择器
语句 说明
title竖线=”head” 匹配title=’head’的元素，或以head-开头
title$=”num” 匹配num结尾
title!=”main” 匹配title不等于main，CSS没有此选择器
title~=”num” 匹配空格隔开的多个属性中的num
title*=”num” 匹配包含num字符串的
$(‘a[title=”head”][class=”top”]’) 这是多属性匹配

过滤选择器

  • jquery里的过滤选择器和CSS3的伪类选择器相似。

过滤器补充

举例 说明
:header 选择标题元素，h1~h6
:animated 选择正在执行动画的元素
:focus 当前被选中为焦点的元素

过滤器对应的jquery方法

方法名 说明
first() 第一个元素,还有last()
not() 非所选择的元素，如$(‘li’).not(‘.red’);
eq() 索引等于某个数，如eq(2)选择第三项

内容过滤器

语句 说明
:contains(“google.com”) 选取含有文本google.com的元素
:empty 选择不包含子元素或空文本的元素
:has(selector) 包含某特征的元素，如$(‘:has(.red)’)
:parent 选取含有子元素或文本的元素

对应的方法

方法名 说明
parent() 选择当前父元素
parents() 选择所有祖先元素
parentsUntil(‘div’) 往上选择所有祖先元素，直到遇到div停止，不包含div

子元素过滤器

伪类 说明
:first-child 获取第一个子元素，对应的最后一个子元素是:last-child
:only-child 只有一个子元素的元素
:nth-child() 获取指定索引的子元素，$(‘li:nth-child(even)’)选择奇数项，:nth-child(3n)隔3选择一个

其他方法

方法名 说明
is() 判断是否是某对象，如$(‘.red’).is(‘li’)是否是li元素
slice(m,n) 选择第m+1项到第n项，可用负数表示倒数，如$(‘li’).slice(2,4)选择第3个和第4个
next() 选择后面一个同级元素，prev()是选择前一个
size() 计算jquery集合对象的元素个数,如$(‘ul’).children().size()有多少个子元素
contents() 返回文本元素，如contents().size()返回文本元素个数
filter() 过滤器，$(‘li’).filter(‘.red,:first’)选择类red和第一个元素

六、DOM操作

常用方法

内部插入

例子 说明
var box = $(‘
节点
‘);		 创建节点

外部插入

例子 说明
after(content); 跟在元素后面插入，成为同级元素，before()表示前面插入

包裹节点

例子 说明
$(‘p’).wrap(‘<div>123</div>’); 变成<div>123<p>haha</p></div>，如有两个<p>则包含两个
unwrap() 移除一层包裹的内容，移两层则用两次
$(‘p’).wrapAll(html) 把所有p用html包裹，不用每个都包裹，html须为DOM对象
wrapInner(html) 内部加一层包裹，即将所有子元素用html包裹

其他节点操作

例子 说明
$(‘div’).clone(true).appendTo(‘body’) 复制div插入到body里面
$(‘div’).remove(‘.box’) 删除div里带box类的元素
$(‘div’).detach().appendTo(‘body) detach()保留事件，重新插入body后事件还存在，而remove()则不保留事件
empty() 清空节点，但节点名称还保留，如剩下<div></div>，内部被清空了
replaceWith(html) 用html替换
